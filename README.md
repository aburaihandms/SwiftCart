1. Null VS Undefined

Undefined:  এটা একটা ভেরিএবোল ডিক্লিয়ার করে কিন্তু তার ভিতরে কোন value দেওয়া হয়না। এটা জাভাস্ক্রিপ্টের default ব্যাবহার।

Null: এটা নিজে থেকে সেট করা যায়। 

2. ==vs===

{==(Double Equal)}: এটা শুধু value চেক করে, type চেক করেনা।

{===(Triple Equal)}: এটা value এবং type দুইটাই চেক করে।
